# 关系模式

**用户** (<u>用户ID</u>, 用户名, 密码, 身份证号, 手机号)

**地区** (<u>地区ID</u>, 省, 市, 区)

**商圈** (<u>地区ID</u>, 区域)
- 外键：地区ID → 地区(地区ID)

**酒店** (<u>酒店ID</u>, 酒店名称, 地区ID, 详细地址, 联系电话)
- 外键：地区ID → 地区(地区ID)

**客房** (<u>客房ID</u>, 酒店ID, 房型, 价格, 剩余数量)
- 外键：酒店ID → 酒店(酒店ID)

**客房图片** (<u>客房ID</u>, <u>图片URL</u>)
- 外键：客房ID → 客房(客房ID)

**客房设施** (<u>客房ID</u>, <u>设施名称</u>)
- 外键：客房ID → 客房(客房ID)

**订单** (<u>订单ID</u>, 用户ID, 客房ID，订单时间, 订单状态, 入住时间, 离店时间, 数量)
- 外键：用户ID → 用户(用户ID)
- 外键：客房ID → 客房(客房ID)

**入住人员** (<u>订单ID</u>, <u>身份证号</u>, 电话)
- 多值属性，依赖于订单
- 外键：订单ID → 订单(订单ID)

**人员** ( <u>身份证号</u>, 姓名)

**评价** (<u>用户ID, 订单ID</u>, 评分, 内容, 评价时间)
- 外键：用户ID → 用户(用户ID)
- 外键：订单ID → 订单(订单ID)
